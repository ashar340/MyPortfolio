The Healthy Theory

MEAN Stack

Current Capstone Project Ionic-Angular based app which enables users to help connect with dieticans/nutritionists nearby to help them achieve their goals bringing them one step closer towards a healthy lifestyle. Data for this application is hosted on a MongoDB Atlas database which connected to the Angular/Ionic client which was can be used cross-platform (Mobile- Android/iOS and Web).

The design analysis and techniques of the project included UML using Visual Paradigm to layout the DB entities before they were put into development, the Software Development Life Cycle methodology, Agile project management using Azure DevOps Boards and tools to manage development phases.

  • Connect/View nearby dietician/nutritionists
  • Make an appointment
  • View Information about the dietician which includes their Specializations, Skills and Services offered
